# Portfolio

Personal portfolio site. HTML/CSS/JS, no frameworks.

## Run locally

Just open `index.html` in a browser. Or use Live Server in VS Code.

## Deploy

Push to GitHub and enable Pages, or drag the folder into Netlify.

## Stack

- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JS
- Google Fonts (Inter, Space Mono)
- FormSubmit for contact form
